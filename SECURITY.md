# Security Policy

## Supported versions

This project is currently in `2.0.0a*` alpha. Security fixes are applied to the latest alpha release only.

## Reporting a vulnerability

Please report security issues privately by opening a GitHub issue with minimal reproduction details and marking it as security-sensitive if available. Do not include private workbooks, credentials, or proprietary business data in public issues.

## Data handling

Excel fixtures committed to this repository must be synthetic or explicitly anonymized and approved for redistribution.
